# simple-protfolio
# 🌐 Hadi Elhaj — Personal Portfolio

A responsive personal portfolio website built with **HTML**, **CSS**, and **Bootstrap 5** as a  project after completing ** introduction Front-End Development Course from meta**.

---

## 📸 Preview

> A clean, colorful portfolio with a fixed sidebar navigation and pastel-themed content sections — showcasing skills, projects, and contact information.

---

## ✨ Features

| Section | Description |
|---------|-------------|
| 🙋 **Hero** | Introduction with name, bio, and a GitHub button |
| 👤 **About Me** | Personal background and developer interests |
| 🛠️ **Skills** | Two-column grid of programming skills |
| 📁 **Projects** | Cards for each project with description and GitHub links |
| 📬 **Contact** | Email and LinkedIn icon links |

---

## 🛠️ Built With

- **HTML5** — Page structure and semantic content
- **CSS3** — Custom colors, layout, and component styling
- **Bootstrap 5.1.3** — Responsive grid system and utility classes

## 🎨 Design Highlights

- **Sidebar navigation** with smooth anchor scroll links
- **Pastel color palette** — warm oranges, greens, and blues per section
- **Circular profile image** using Bootstrap's `rounded-circle` utility
- **Two-column skills layout** using Bootstrap grid
- **Icon-based contact section** with email and LinkedIn links

## 🏃 How to Run Locally

```bash
git clone https://github.com/hadialhaj/simple-protfolio.git
cd simple-protfolio/prot
# Open index.html in your browser
```

> No build tools or dependencies needed — just open `index.html` directly.
